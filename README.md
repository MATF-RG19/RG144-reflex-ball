# RG144-reflex-ball
## Testiraj svoje reflekse

_Stubovi se podizu vremenom a zadatak igraca je da pritiskom na odgovarajuci taster spusti podignuti stub._

## Kontrole

| Taster        | Funkcija                |
| :-----------: |:-------------:          |
| G             | Pokretanje igre         |
| S             | Zaustavljanje igre      |
| R             | Vracanje na pocetak     |
| 1-9           | Spustanje stuba         |
| ESC           | Izlazak iz igre         |
