# RG144-reflex-ball
## Testiraj svoje reflekse

_Pritiskom na tastere od 1-9 se spustaju odredjeni stubovi._

## Kontrole

| Taster        | Funkcija                |
| :-----------: |:-------------:          |
| G             | Pokretanje igre         |
| S             | Zaustavljanje igre      |
| R             | Vracanje na pocetak     |
| 1-9           | Spustanje stuba         |
| ESC           | Izlazak iz igre         |
